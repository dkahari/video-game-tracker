# Video Game Tracker

Video Game Tracker is a personal Progressive Web App for tracking games you have bought, played, completed, or dropped. This project is being built as a learning and portfolio app using Vue, TypeScript, Material Design, and PWA concepts.

## Project Status

Planning and architecture are currently in progress. The implementation setup has not started yet.

## Planned Stack

- Vue 3
- TypeScript
- Vite
- Vuetify
- Vue Router
- `localStorage` for initial persistence
- PWA support

## MVP Goals

- Track a personal game library
- Add, edit, and delete games
- Track status and simple progress
- View dashboard summary stats
- Persist data locally across reloads
- Support installable PWA basics
